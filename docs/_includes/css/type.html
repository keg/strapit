Typography

Built with Less

The typographic scale is based on two Less variables in variables.less: @font-size-base and @line-height-base. The first is the base font-size used throughout and the second is the base line-height. We use those variables and some simple math to create the margins, paddings, and line-heights of all our type and more. Customize them and Strapit adapts.

Headings

All HTML headings, <h1> through <h6>, are available. .h1 through .h6 classes are also available, for when you want to match the font styling of a heading. If you want the styling to be applied inline, add the class .inline to the .h* class

Body copy

Strapit's global default font-size is 14px, with a line-height of 1.618. This is applied to the <body> and all paragraphs. In addition, <p> (paragraphs) receive a bottom margin of half their computed line-height (10px by default).

Abbreviations

Stylized implementation of HTML's <abbr> element for abbreviations and acronyms to show the expanded version on hover. Abbreviations with a title attribute have a light dotted bottom border and a help cursor on hover, providing additional context on hover.

Located in type-rare.less

Abbreviations and the style shown here afterward are kind of rarely used. Because of that, we separated these styles into another Less stylesheet named type-rare.less just in case yuou'd like to turn these off in final complication.'
