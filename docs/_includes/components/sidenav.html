Side nav

The side nav is useful for sections of either a site or a page. It can also be used in off-canvas applications for small devices. Developed by Arnold Daniels at Jansy Bootstrap.

Default Side nav

Side nav is a vertical navigation component. By default it shares it look and feel with the side nav component.

Custom width

The side nav is 300px wide by default. You can change this by customizing the @sidenav-width variable or by setting the width of .sidenav in your CSS.

Make side navs accessible

Be sure to add a role="navigation" to every side nav to help with accessibility.

Body padding required

The fixed side nav will overlay your other content, unless you add padding to the left or right of the <body>. Try out your own values or use our snippet below. Tip: By default, the side nav is 300px wide.

{% highlight css %} @media (min-width: 992px) { body { padding-left: 300px; } } {% endhighlight %}

Make sure to include this after the core Strapit CSS.

Off canvas

With the offcanvas plugin, you can hide the side nav off canvas. This is especially useful for screens with a small viewport.
